Server Resources

Dev Server: http://localhost:3000/api

Test

  • GET: api/test/ex4/{name}
  • GET: api/test/
  • POST: api/test/ REQUEST BODY: { "id": number "nome": string }
  • PUT:api/test/{id} REQUEST BODY: { "nome": string }
  • PATCH: api/test/{id} REQUEST BODY: { "nome": string }
  • DELETE: api/test/{id}

User

  • GET: api/usuario
  • POST: api/usuario REQUEST BODY: { "id": number "nome": string, "idade": number, "cargo": number "senha": number }
  • PUT: api/usuario/{id} REQUEST BODY: { "nome": string, "idade": number, "cargo": number "senha": number }
  • DELETE: api/usuario/{id}
